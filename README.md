## 🎯 あなたのミッション

1. このリポジトリを Fork してください。
2. ローカル環境または Codespaces で開いてください。
3. テストを実行してみましょう：
    - Mac/Linux:
      ```sh
      sh run_tests.sh
      ```
    - Windows:
      ```
      run_tests.bat
      ```
4. テストが失敗した理由を調べましょう。
5. `src/` や `test/` の中のコードを読んでみてください。
6. GitHub上の Pull Request やコミット履歴を調べてみてください。
7. 怪しい変更を発見し、コードを修正しましょう。
8. 修正できたら Pull Request を作成して提出してください。
